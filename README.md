# 🎬 Netflix Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Netflix Dashboard** built using **Microsoft Power BI**. It provides insights into Netflix's content library, including movies and TV shows, by analyzing ratings, genres, release years, countries, and content types.

The dashboard helps users explore Netflix's catalog through interactive visualizations and filters.

## 🚀 Features

- 📈 Total Movies, TV Shows, and Overall Titles
- 🎭 Distribution of Content by Genre
- ⭐ Content Count by Rating
- 📅 Release Trend Over the Years
- 🌍 Country-wise Content Distribution
- 🎥 Movie vs TV Show Comparison
- 🔍 Interactive Filters:
  - Show Type
  - Release Year
  - Country
  - Duration/Seasons

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
- Netflix Dataset (CSV)

## 📂 Dataset

The dataset contains information such as:

- Show ID
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description
- Content Type (Movie/TV Show)

---

## 📈 Dashboard Insights

- Netflix contains approximately **9K total titles**.
- Around **6K Movies** and **3K TV Shows**.
- Drama is the most common genre.
- TV-MA is the most frequent content rating.
- Significant growth in Netflix content production occurred after 2015.
- Content is distributed across multiple countries worldwide.

---

## 📷 Visualizations Included

- Bar Chart (Genre Distribution)
- Column Chart (Ratings)
- Area Chart (Release Year Trend)
- Donut Chart (Movies vs TV Shows)
- World Map (Country Distribution)
- KPI Cards
- Slicers for Interactive Filtering
